# 题目描述
输入一个链表，输出该链表中倒数第k个结点。
# 程序代码
时间复杂度O(n),空间复杂度O(1)
```
/*
public class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}*/
public class Solution {
    public ListNode FindKthToTail(ListNode head,int k) {
        int i=0;
        for(ListNode p=head;p!=null;p=p.next){
            i++;
        }
        int j=0;
        for(ListNode p=head;p!=null;p=p.next){
            if(j+k==i){
                return p;
            }
            j++;
        }
        return null;
    }
}
```
