# 题目描述
大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。(n<=39)
# 方法一 循环求和
f(n)=f(n-1)+f(n-2)，时间复杂度O(n)。
# 方法一 程序
```
public class Solution {
    public int Fibonacci(int n) {
        if(n<=1){
            return n;
        }
        int f0=0,f1=1;
        int fn=0;
        for(int i=2;i<=n;i++){
            fn=f0+f1;
            f0=f1;
            f1=fn;
        }
        return fn;
    }
}
```
# 方法二 动态规划
如果求f(n)的值，需要创建一个容量为n+1的数组，时间复杂度O(n)。
# 方法二 程序
```
public class Solution {
    public int Fibonacci(int n){
        if(n<=1){
            return n;
        }
        int[] f=new int[n+1];
        f[0]=0;
        f[1]=1;
        for(int i=2;i<=n;i++){
            f[i]=f[i-1]+f[i-2];
        }
        return f[n];
    }
}
```
